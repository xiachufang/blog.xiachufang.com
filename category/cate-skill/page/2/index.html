「窍门」吃皮蛋别忘了放姜醋汁

2011年1月25日 小饭

皮蛋又叫松花蛋，有许多种做法，可以单独成为一道凉菜，也能与黄瓜小炒或是相伴成汤。通常情况下有皮蛋的菜里都会有姜、蒜、醋、辣椒等比较抢味的配料。这是因为皮蛋多是用鸭蛋腌制的，腌制时放入了茶叶、石灰，其中含有的有机茶酚单宁及产生的氢氧化钠等物质渗透到蛋里，加工时蛋白质分解产生出一定的氨气，使得皮蛋有一种碱涩味。

用姜末和米醋配成的姜醋汁中含有姜辣素和有机酸，不仅能去腥，还能中和蛋白体中含有的碱性物质，有效去除碱涩味。此外，姜醋汁有很好的解毒、杀菌作用，还能促进胃液分泌，增进食欲、帮助消化。不仅是皮蛋，各类白灼、清蒸海鲜以及凉拌菜都可搭配姜醋汁。

PS：皮蛋最好用棉线切割。

姜醋汁的制作

[姜醋汁1]
材料：姜末20克、蒜末15克、蜂蜜少许、糖3大匙、白醋4大匙
做法：将所有食材混合均匀即可。

[姜醋汁2]
材料：米醋7汤匙、白糖2汤匙、姜末适量
做法：将所有食材混合均匀即可。

[姜醋汁3]
材料：生姜切末约2汤匙、糖约3汤匙、醋约4汤匙、少许味精或鸡精、2汤匙开水
做法：微波炉加热一分钟，用筷子搅至糖完全溶化即可。

「窍门」厨房变干净，食醋来帮忙

2011年1月22日 小饭

喝醋好处多多大家都知道，健脾开胃、美容养颜等等。可是醋也是我们清洁厨房的好帮手你知道吗？新年啦，用醋给你的厨房做套美容餐吧！

[地面]
厨房和饭厅因为做饭及用餐，地板可能会有一层腻腻的感觉。如果倒上一些食醋，再用拖把用力拖几遍，地面就能光亮洁净。

[铁锅]
用新的铁锅烧饭会发黑。不妨把铁锅放于火上烧烫，倒入200~250克醋，待发出声响时，用刷子擦拭、洗净即可。

[铝器]
铝制器皿用旧了，可以用醋涂抹，干后用水洗刷即可恢复光亮。

[铜器]
铜制品如果有了锈斑，用醋擦洗能立即洁净。

[陶瓷]
新买的陶瓷餐具，也许对表面装饰和色彩的安全度不太放心，但因为太喜欢又无法割舍。使用之前，用浓度为4%的食醋水浸泡并煮沸，有助于去除大部分的有毒物质。

[隔热垫]
用久了的隔热垫会脏、不好看，蘸少许醋清洗能变干净。

[异味]
如果厨房有异味，可以把一口锅放置于火上，在锅里滴少许食醋让它散发，厨房的异味也能消除。（注意控制用量，太多了空气会有馊掉的感觉。）PS：橘皮或是香蕉皮放在火上烤效果也不错。

「窍门」厨房清洁，妙用食盐

2011年1月19日 小饭

过年总是意味着辞旧迎新，这时不论是办公室还是家中，大扫除总是必修课。厨房清洁应该是最难的科目了。一些强力的洗涤剂也许家中并没有备着，但某些家里一定有的东西这时能帮上大忙，比如盐。盐的清洁能力你可别小觑，能被用到的地方可能是你想都没想过的。

[水槽]
厨房的水槽很容易有污垢，要清洗而家里又没有去油污剂时，可以抓两把盐，平均地撒散，过一会儿后用热水冲刷。水槽不易洗刷的小角落，可以请废旧的牙刷来帮忙。

[白瓷砖]
厨房的白瓷砖有了黄渍，可用布蘸精盐，每天擦2次，连续2~3天后，再用湿布擦几次，白瓷砖能洁白如初。

[砧板]
砧板日常的消毒可以巧用食盐。每次用完后刮干净板上的残渣，在上面撒盐（一周一次），既可以杀菌又能防止砧板干裂。

[铝锅]
常用布蘸盐擦拭铝锅表面可光洁如新。

[刀]
用盐水磨刀，既省力又锋利，还能延长使用寿命。方法是先把刀在盐水中浸泡20分钟，再在磨刀石上磨，边磨边淋盐水，磨好后用清水冲洗干净，涂上一层油，就不易生锈了。

[茶壶/茶杯]
茶壶和茶杯时间长了内部就容易积出茶锈。这时用湿布蘸点盐末去擦，茶锈很容易就能被去掉。一些不易擦到的边边角角可以借助镊子或是筷子的力量。

「新手」这样煮牛奶，营养不流失

2011年1月18日 小饭

天气越寒冷，早餐的热鲜奶就越是显得诱人。美白滋润又温暖，活力一整天为营养加油。不管是在家中还是办公室，只要有设备大家都愿意把牛奶加热了再喝。可是你知道吗，如果对牛奶使用的加热方法不当，营养会严重流失。

加热牛奶的正确方法，只需要记住一条原则，用旺火加热，煮熟后加糖。因为用文火煮，会使牛奶中所含的维生素被空气氧化而损失。而在高温下，牛奶中所含的赖氨酸会与糖形成一种叫糖基赖氨酸的有害物质。所以煮牛奶前不要加糖，而要在煮熟后，晾温时加糖。

首选：煤气灶或电磁炉加热
明火以及电磁炉加热，升温速度快且加热均匀，保住了维生素营养不流失。煮沸后，马上熄火，晾温后就能喝了。
家中如果有一只迷你不锈钢牛奶锅（要玻璃盖的）最好了！为什么要是玻璃盖的呢？煮牛奶很容易沸腾溢锅，告诉大家一个小窍门，煮的时候注意观察牛奶变化，当牛奶将要沸腾出现第一个“气泡”的时候，立即关火。玻璃盖可以方便我们观察锅内的牛奶。

次之：微波炉加热
都市白领们几乎是离不开微波炉的，办公室的茶水间里加热三明治、牛奶快捷又方便。一般说来250ml的牛奶微波炉高火加热一分钟就好。但微波炉加热的牛奶通常是“冰火两重天”的效果，因为受热不均匀一杯牛奶往往外热内冷。喝之前最好搅拌一下不会烫嘴。而且如果用微波炉加热牛奶时间过长，牛奶中的蛋白质受高温作用会变成凝胶状态产生沉积物影响口感。所以微波炉加热牛奶是次选。

加热牛奶需要注意的事项
1.无论是煤气灶、电磁炉还是微波炉，都不能为了偷懒不刷锅不洗杯子而连同牛奶包装一并加热，会产生有毒物质。
2.高温加热牛奶的时间不宜过长，牛奶长时间煮沸会产生有害物质。
3.不宜采用铜质器皿加热牛奶。

「窍门」怎样去除萝卜异味

2011年1月13日 小饭

冬吃萝卜夏吃姜，冬天正是食用被誉为“小人参”的萝卜的最佳时节。萝卜营养丰富、脆嫩多汁，可是如果不经过一些“加工处理”，吃起来会有一种“怪怪的味道”。几招简单、实用的小窍门教你有效去除萝卜的异味。

开盖煮去味法
煮萝卜时，打开锅盖煮可以去除异味。

焯水去味法
萝卜在烹饪前焯水过能去除异味。萝卜有辣味和涩味是由于萝卜内含有名为硫醇和黑芥子甙的物质，这两种物质经烫焯后能转变成可挥发的芥子油，不仅除去了萝卜中的辣味、涩味，同时还能使萝卜中的部分淀粉转化成葡萄糖而产生微甜味、鲜味。

食盐去味法
在烧萝卜前，在切好的萝卜上撒适量盐，渍一下滤去萝卜汁，可减少萝卜苦味。

食醋去味法
如果是蒸吃萝卜，可先将其切碎，按300：1的比例放入食醋再上锅蒸，异味可消除。

小苏打去味法
将萝卜切碎，按照萝卜：小苏打=300:1的比例撒上小苏打拌匀煮熟，这样烧出来的萝卜也没有异味。

「新手」做盖浇饭好吃的技巧

2011年1月11日 小饭

盖浇饭，是将配料烹制后不与米饭同炒，直接浇在米饭上即可。盖浇饭的品种非常多，有些地方还会有专门供应盖浇饭的餐厅，咖喱牛肉盖浇饭、红烧鸡块盖浇饭、番茄鸡蛋盖浇饭等等满满地占据了菜单，看得人眼花缭乱每一种都想吃而难以抉择。不妨自己在家做吧！掌握了做盖浇饭好吃的技巧，冰箱里现有的材料都能很好地被运用进去，经常换着花样，尝试各种菜色，懒人也会有好口福的。

盖浇饭的重点是米饭上的菜肴要鲜香、味浓、有勾芡。清淡味寡的菜不下饭，不适合用来做盖浇饭；适量的勾芡可以有浓稠的汤汁来拌饭，看上去让人更有食欲而米饭吃起来也更加爽滑，勾芡的汤汁还有保温食材的功效。

几款美味易做的盖浇饭。
[酱爆鸡丁盖浇饭]
材料：鸡脯肉1块、胡萝卜1根、黄瓜1根、青椒1个、花生米、甜面酱、豆瓣酱、葱白、蒜头、鸡蛋（取蛋清）
做法：1. 黄瓜胡萝卜去皮切丁（大小随意），青椒去蒂切小块，蒜和葱白均切末，备用；
2. 鸡脯肉切丁后，取蛋清，1勺料酒，少许盐，拌均匀，腌制5分钟；
3. 锅中倒适量油，烧热后，下胡萝卜丁，蒜末葱末煸炒2分钟，然后下入黄瓜和青椒炒1分钟；
4. 将腌制过的鸡丁下入，不停翻炒以防粘锅，至鸡丁变色，粒粒分明；
5. 将鸡丁等食材用锅铲推至一旁，将1勺甜面酱，1勺豆瓣酱倒入锅中央，稍稍加热后，和食材一起翻炒均匀，再倒入花生米，略炒1分钟左右就行了，盛起浇在热米饭上即可。

[家常卤肉饭]
材料：五花肉、洋葱头、鸡蛋、油菜心、姜末、生抽、老抽、八角、料酒、冰糖、五香粉、胡椒粉、盐
做法：1、买来的五花肉，洗净，切成小丁，切的小一些。
2、洋葱或是红葱头，洗净后切碎，葱姜切碎末。
3、炒锅内倒入适量油，先爆香洋葱碎。
4、继续炒香，直至金黄。
5、这时倒入五花肉丁炒翻。
6、炒至肉色变白，然后加入生抽、老抽、八角、料酒、五香粉、冰糖，胡椒粉，翻炒均匀。
7、加入适量温水，不用太多，没过五花肉1-2cm即可。
8、另取一锅水煮蛋，捞出去皮待用。
9、烧开水将小油菜心焯熟，捞出待用。
10、大锅中的肉大火改小火慢炖50分钟后，这时别忘了加入水煮蛋卤蛋入味，炖的时候火要小，汤汁也不要收的太干，有汁才会好吃，最后装盘盖在米饭上撒香葱末。

[咖喱鸡翅盖浇饭]
材料：鸡翅3块、土豆1个、胡萝卜2根、咖喱粉、蒜头、牛奶、料酒、黑胡椒、生抽、
做法：1.原料备好洗净后，将鸡翅入水焯一下捞起，加料酒，黑胡椒，生抽腌制一会；
2.土豆、胡萝卜切块，蒜切末（这部分蔬菜可自由添加，其实加点洋葱或苹果，味道也不赖）。
3.锅中倒油，爆香蒜末后，倒入腌过的鸡翅，煸炒；
4.鸡翅变色后，下入土豆和胡萝卜同炒，动作轻柔些，土豆易碎；
5.锅中倒入一小碗牛奶，加入咖喱粉，少许盐，大火烧开；
6.烧开后转小火，烧5分钟左右，加黑胡椒粉，收汁，可根据喜好留多少汤汁，盖在饭上即可。

[黑椒猪排饭]
材料：猪里脊肉、鸡蛋、面粉、米饭、西兰花、蒜、香油、盐、蚝油、生抽、黑胡椒碎、白糖、水淀粉
做法：1.饭蒸好后打散一个鸡蛋。
2.这款猪排不用面包糠、不用油炸，准备一点面粉再加一个鸡蛋就可以了。
3.猪里脊肉，取一小块用肉锤或刀背把它拍打成大大的薄片，然后加腌肉的调料腌制一会。这样猪排既嫩又入味。
4.在腌过的猪排表面裹上一层面粉，再抖去多余的面粉。
5.再放进蛋液里浸泡15-20分钟，让猪排表面的面粉充分吸收蛋液。
6.准备一颗西兰花，掰成小朵洗净，放入沸水中焯熟，放入大容器，加盐、蒜茸、香油拌匀就可以吃。
7.把腌好的猪排放入煎锅煎熟，炒菜量的油，中小火煎就可以了。
8.盛上米饭，猪排切条，放在米饭上。
9.最后再做一点黑椒汁，两人份：两勺蚝油、两勺生抽，两勺清水适量的黑胡椒碎，一点点糖，煮开后勾一点点薄芡，浇在猪排和米饭上就可以开饭了。

[咖喱牛肉烩饭]
材料：米饭2碗、牛腩300克、洋葱1/4个、马铃薯1个、胡萝卜半根、葱段少许、姜片3~4片、咖喱粉1小匙、咖喱2/3块、料酒、盐、香叶1小片
做法：1.牛腩洗净，汆烫，再加葱段、姜片、料酒和香叶炖1.5-2小时，最后可加少许盐调味。
2.将胡萝卜、土豆、洋葱都洗净，切块，土豆切块后用清水冲洗一下，再控去水分。再准备一点豌豆和玉米粒，胡萝卜和豌豆放入小汤锅中略煮，土豆块放入油锅中炸熟，捞出备用。
3.炒锅中加入2大匙油，炒香洋葱和咖喱粉，再加入牛肉，牛肉汤，盐略煮，然后加入胡萝卜，豌豆，玉米粒，土豆块，最后放入咖喱块，再煮5分钟，吃时淋在米饭旁。炸土豆可以留一部分，等到吃的时候撒在咖喱上。

[番茄滑蛋牛肉饭]
材料：米饭1碗、番茄2个、牛里脊肉100克、鸡蛋1个、葱小半根、姜片3-4片、盐1-4小匙、料酒2小匙（分两次放，一次1小匙）、淀粉1/4小匙、色拉油1大匙、清水半杯、酱油1小匙、白糖1/4小匙、水淀粉适量、香油适量
做法：1. 把牛肉切片，番茄切成大的滚刀块，葱切段，姜切片，鸡蛋打入碗中备用。
2. 牛肉切好后放入容器，加入盐，料酒，淀粉，色拉油拌匀，放在一边腌30分钟左右。
3. 锅内加入少许油，烧热后倒入牛肉过油，变色后捞出沥油。
4. 锅内加入一大匙油，爆香葱段和姜片，加入番茄块翻炒一下，放入清水，酱油，料酒，白糖，少许的盐，煮滚后勾薄芡，加入牛肉片，淋上蛋液，最后淋入少许香油即可出锅。吃的时候把米饭盛在平盘上，淋上番茄滑蛋牛肉就好了。

[咖喱鸡肉饭]
材料：鸡腿肉200克、百梦多咖喱2块、土豆150克、胡萝卜100克、洋葱75克、鲜香菇50克、花生油25克、料酒10克、生抽10克、姜片少许、盐少许
做法：1.将鸡腿肉剔下来（一根鸡腿大约200克左右）洗净切成小块放入碗中，加入料酒、生抽、盐、姜片腌渍1小时左右;
2.上海好侍百梦多咖喱，超市有售;
3.土豆、胡萝卜去皮洗净后切成小块，洋葱、鲜香菇洗净切小块，咖喱2块；
4.炒锅倒入适量油，烧热后放入鸡肉煸炒至变色；
5.放入洋葱炒几下接着放入土豆、胡萝卜、香菇翻炒一会；
6.倒入开水或清水，刚刚没过食材为好。大火烧开转中火，煮一会感觉食材熟透后放入咖喱块。转小火用铲子不断搅拌，加快咖喱的溶解，以免糊底。一定要尝一尝滋味，如果淡就再加一点盐。待汤汁变浓稠即可。把菜肴浇在饭上即可。
