「情调」有趣！水果性格说

2010年7月1日 Michelle

看过《全城热恋》的筒子们应该都会对开头的那些挥汗如雨的镜头印象深刻，M所在的城市夏天虽然没有那么热，但也算得上是“火炉”一个。所以每到夏天，就处于长期脱水的状态，一杯接着一杯地喝，还真有那么点“骆驼祥子”的状态……说到饮食方面，水果是M的夏天大爱，不光味道清新爽口，补充水分和能量，还只有很少很少的热量。哇咔咔！所以说夏天也是个“享瘦”的季节呢。JMS都一起加油吧！

西瓜

以前曾经看过一篇文章，说是从你爱吃的水果能看出你的性格，M自测了一下，貌似还蛮准的。M最爱吃的水果是西瓜。这是夏季的主打水果，含有大量的水分，糖分含量仅占5%到10%左右，主要为葡萄糖、果糖和蔗糖。几乎不含淀粉，采摘后贮藏期间也会因为双糖水溶解为单糖而降低。有些地方还会把瓜皮加工成西瓜酱食用。在中医上以西瓜汁和西瓜皮入药可以清热解暑。当然啦，也有些人是不适合吃西瓜的呢。

西瓜fan的性格：脾气很好，不爱抱怨，不与人争吵，关心别人胜过爱护自己，但是缺乏原则性。

香蕉

M身边最Sweet的朋友就是个典型的“香蕉控”，一年四季都要吃香蕉，每次去水果店，香蕉是必买之物。大概也得益于此，那丫头从不便秘。

香蕉是让人心情愉悦的水果，能够帮助内心软弱、多愁善感的人驱散悲观、烦躁的情绪，保持平和、快乐的心情。这主要是因为它能增加大脑中使人愉悦的5—羟色胺物质的含量。研究发现，抑郁症患者脑中5—羟色胺的含量就比常人要少。

香蕉fan的性格：外表坚强、内心软弱，有些多愁善感，在意别人对自己的评价。因此不要封闭自己的内心，走出自己的小小角落，跟更多的人接触一下，也许会发现世界并没有那么可怕哦！

草莓

很多女生都喜欢草莓，不光是爱那种水果，连草莓眼罩、小草莓空调被、草莓口香糖也是一堆一堆的。草莓属于低矮草茎植物，所以生长过程中极易受到污染，因此，吃草莓简单，但是洗草莓确实一件超级要耐心的事情呢。在挑选好草莓之后，先摘掉叶子，用活水冲洗，然后用盐浸泡5到10分钟，最后再用凉开水浸泡1到2分钟。经过这么多分钟的折腾，你才能将这颗营养丰富的“活维生素丸”吃下哦。

草莓fan的性格：乐观开朗，非常有自信，懂得享受生活，但做事有时候缺乏耐心。所以不如多洗洗草莓吧，在别的童鞋要吃的时候也主动帮着洗洗，也许能多多磨练出耐心来哦。嘿嘿！

葡萄

“吃葡萄不吐葡萄皮儿，不吃葡萄倒吐葡萄皮儿。”如果你嫌这则绕口令太简单了，不如先念个5遍试试看。小时候觉得这个绕口令太荒谬了，长大了才知道，最健康的吃葡萄的方法其实是不吐葡萄皮儿也不吐葡萄籽，因为葡萄皮儿和葡萄籽比葡萄肉更有营养。据说红葡萄酒之所以比白葡萄酒有营养就是因为它是连皮儿一块儿酿造的，M的母亲也超爱自己酿葡萄酒的，自家的葡萄酒什么添加剂都没有，香醇的口感让人欲罢不能。研究人员还发现，葡萄籽中含有丰富的增强免疫、延缓衰老物质的OPC，而且进入人体后可以达到高于85%的吸收量哦！

葡萄fan的性格：善于交际，组织能力强，但是不会锋芒太露，懂得保护自己。但另一方面，生活比较懒散哦。

樱桃

M最早买樱桃吃完全是觉得生日蛋糕上那小小一颗的樱桃实在是不过瘾，樱桃中含铁很高，是特别适合女性吃的水果食物，有补虚养血的功效。美国科学研究人员发现吃樱桃能明显减轻疼痛感。冬天干燥，口中容易出现异味，挤出樱桃汁，加水稀释后涮口，就能帮你消除这个烦恼哦。

樱桃fan的性格：善于理财，懂得持家，但是容易寂寞，害怕孤独。

梨

梨的水分充足，富含维生素和微量元素，能维持细胞组织的健康状态，帮助器官排毒、净化，还能软化血管，促进血液将更多的钙质运送到骨骼。因此梨是令人生机勃勃、精力旺盛的水果。科学研究证明，吃梨时一定要细嚼慢咽才能更好地吸收哦。

梨子fan的性格：有才华，精力充沛，对于任何认定的事情都不轻言放弃，但是有时候过于倔强，有顽固之嫌哦。

橘子

哈佛医学院的专家建议经常情绪激动的人应当常吃橘子来降低患心脏病、高血压和中分的危险。但是注意不要超过4个，否则可能上火，如长口疮等等。

橘子fan的性格：情感丰富，具有亲和力，但有时非常情绪化，态度飘忽不定，让人捉摸不透。

柚子

说到柚子，江西南康的柚子实在是让人印象深刻，M每次冬天去都要买很多回来。果肉碧绿，汁多味甜、质脆爽口、果大耐贮藏。柚子是保证人体健康，使心血管系统健康运转的水果。含有丰富的果胶能降低低密度皮脂蛋白，减轻动脉血管壁的损伤，维护血管功能，预防动脉硬化和心脏病。科学研究表明，吃8只柚子能明显促进运动中受伤的组织器官恢复健康。

柚子fan的性格：身体很健康，充满阳光气息，有很好的运动细胞，但是自我意识太强，容易急躁。

苹果

“一日一苹果，医生远离我。”每天吃少量的苹果就能预防多种疾病，还能让人产生饱腹感，难怪很多JMS都把苹果当做减肥餐来食用。美国癌症研究中心特别建议人们常吃苹果来预防癌症，因为其中含有的丰富的天然抗氧化剂能够有效消除自由基，降低癌症发病率。

苹果fan的性格：为人务实，思维沉稳，做事有计划，能冷静思考，不怕艰苦，但是自尊心强，有些守旧。

李子

李子美味多汁，清肝热，活血脉，据说有美容乌发的神效。但是李子食用过多容易生痰，容易损坏牙齿，体质虚弱者应少食。根据前人经验，味苦的李子和漂浮在水中的李子是有毒的，要小心慎食。

李子fan的性格：不易与人和睦相处，不能忍受批评，不喜欢唠叨，爱找茬儿。（此观点不代表下厨房言论^_^*）

桃子

桃子的果肉中含有丰富的蛋白质、脂肪、糖分、钙、磷、铁、维生素和大量的水分，因此想减肥的JMS还是少吃为妙。桃子对慢性支气管炎、支气管扩张症、肺纤维化、肺不张、矽肺、肺结核等出现的干咳、咳血、慢性发热、盗汗等症，可起到养阴生津、补气润肺的保健作用。桃子富含胶质物，这类物质到大肠中能吸收大量的水分，能达到预防便秘的效果。古代医书上说，将桃子晒成干，经常服用，能起到美容养颜的作用。只是桃干的含糖量过高，用开水与少量绿茶或花草茶冲服就会好很多，而且口感还很独特哦！

桃子fan的性格：与周围的大部分人在心理上都是趋同的，是一个善于交际的人，拥有很多的朋友，但是有的时候在困难面前总会觉得有心无力。

有些筒子们也许并不钟情于一种水果，而是根据季节或心情变化自己的选择。无论如何，吃得开心，吃得健康，那才是饮食的初衷……

「常识」水果怎么吃？（究极版）

2010年6月2日 Michelle

这两天M看到一则新闻，说最近几天辽宁省鞍山市几家医院连续接诊了因为空腹吃山楂、柿子而在胃里形成结石的病患。而且其中的一位女士的经历真的只能用“OMG”来形容。她曾空腹吃了两个大柿子，仅仅只过了三四天，这位女士的胃里就已经形成了一个蛋黄大小的胃石。

在如今“睡觉死”、“喝水死”、“冲凉死”屡现江湖的年代，吃个水果居然也能折腾到医院去，M不经要小小地感叹一下。其实，大家都知道水果的好处，丰富的矿物质、维生素、纤维素、不易发胖的天然糖分以及爽滑的口感都让人欲罢不能。可是水果要什么时候吃，如何选择适合你的水果，很多筒子们确实不知道的。

夏天来临，在这个水果当道的季节，M特备整理了一些水果的小常识，和大家一起分享一下水果要怎么吃。

选择恰当时机

从去年起物价就开始明显上涨，蔬菜都进入“元时代”了，水果更是涨价涨得惊人，M所在的城市普通的黄瓜涨到1.5元一斤，这还是平均价，大部分时候几乎有增无减，水果黄瓜更是涨到2.8元。所以，选择恰当的时候吃水果，让水果的营养和充分的吸收，这才是物价上涨时期的明智之举。

很多人把水果当做饭后甜点吃，其实这样做实在没什么道理，反而不利于消化和营养的吸收。吃水果的黄金时期是两餐之间，这样可以使营养收益最大化。因为水果的养分是水溶性的，容易被吸收，如果在两餐之间吃水果，肠胃可以很快地消化吸收。想减肥的JMS注意了，饭前吃水果可以让胃部产生满足感，这样吃正餐的时候就会自然地减少摄入量。不过注意咯，有些水果比如开头提到的柿子和山楂是不能空腹吃的哦！

不能空腹吃的水果

1、山楂、橘子

很多人认为山楂、橘子这一类水果酸酸甜甜的很开胃，适合饭前吃，这是错误的认识。因为其中含有大量的有机酸、果酸，具有行气消食的作用，如果空腹食用，会使胃酸猛增，对胃粘膜造成不良刺激，引发胃胀满、泛酸，还会增加饥饿感加重胃病。

2、柿子、番茄

柿子和番茄中含有丰富的柿胶粉、胶质、果胶、鞣酸和鞣红素等物质，具有很强的收敛作用。平时的时候吃对身体是没有影响的，但在空腹的状态下，胃里胃酸分泌较多，容易和这些收敛物质融合形成难以溶解的硬块，小硬块可以随着新陈代谢排除体外，但是如果形成了较大的硬块，就容易引起胃柿结石症。

3、香蕉

M曾经看过一本减肥杂志上说早餐只吃香蕉和水瘦身效果明显，并且亲自尝试过一段时间，结果吧胃折腾得奇奇怪怪的，现在想起还是内牛满面啊！香蕉并不适合空腹食用，因为香蕉中含有丰富的糖和纤维素，但在空腹时，肠胃里几乎没有可供消化的食物，这个时候吃香蕉会加快肠胃运动，促进血液循环，增加心脏负荷，严重的时候还会导致心肌梗塞呢。

4、甘蔗、荔枝

这类水果糖分很高，空腹食用会刺激胃粘膜，导致胃痛、胃胀。据说曾经有人在空腹时吃荔枝突然昏迷，医生说是体内突然渗入过量高糖分而引发的“高渗性昏迷”。专家建议，在吃荔枝前后，可以先喝一点盐水、凉茶、绿豆等，这样既不会引发昏迷，也不容易上火哦！

5、菠萝

菠萝里含有丰富的蛋白分解酵素，如果空腹吃，很容易造成胃壁受伤。像M一样喜欢吃菠萝的筒子们不如时刻念叨一下，千万不要空腹吃哦！

选择适合你的水果

很多食物都有酸碱性之分，水果也不例外，大部分的水果都属于亚酸性的，它们的酸碱性取决于水果中所含矿物质的类型。富含粗纤维、钾、钠、钙、镁的水果是低酸性水果，富含氯、磷、硫的水果则是高酸性水果。

高酸性水果：李子（PH值2.5）、杨梅（PH值1.5）、柠檬（PH值2.0）、橙子（PH值2.8）、草莓（PH值3.0）、莲雾（PH值2.8）

中酸性水果：梨子（PH值3.5）、木瓜（PH值4.5）、苹果（PH值4.9）、葡萄（PH值4.5）、桃子（PH值4.2）、菠萝（PH值4.0）、番茄（PH值4.8）

低酸性水果：香蕉（PH值5.1）、西瓜（PH值6.6）、荔枝（PH值5.0）、石榴（PH值5.0）、水蜜桃（PH值6.8）

除了酸碱值，我们在挑选水果的时候还要考虑自己的体质哦。体质燥热的筒子们适合吃梨子、香蕉、西瓜、香瓜等偏寒性水果;体质偏寒者宜吃荔枝、龙眼、樱桃、栗子等;心肌梗死、中风的病人宜吃香蕉、橘子、桃等帮助消化的水果，不宜吃柿子、苹果等含鞣酸较多的水果;冠心病、高血脂病人宜吃山楂、柑橘、柚子等含维生素C和尼克酸的水果，能降低血脂和胆固醇。

应季的才是最好的

科技的发达让人们一年四季都能吃上心仪的水果，但是这样的反季水果真的好吗？一年四季交替，水果也原本是交替成熟的，在不同的季节吃不同的水果是我们的老祖宗几千年来的惯例。如今的化学物质或催熟或保鲜，这些催熟剂、保鲜剂、膨大剂或激素都是通过人工合成的化学物，具有一定的毒性，吃进去会加重肝肾的负担，经常食用还会损伤肝肾功能。许多人工激素还会干扰人体正常的内分泌功能，影响生长发育或产生其他疾病，因此反季水果有噱头，挑选食用需谨慎！

春季水果：枇杷 草莓 香瓜 桃子 樱桃 李子 杨桃

夏季水果：菠萝 西瓜 荔枝

秋季水果：梨 苹果 卢柑 枣子 葡萄 香蕉 猕猴桃 木瓜

冬季水果：桔子 甘蔗 橙子

食用水果应遵循的原则

1、不过量原则

很多筒子们都认为水果既能迅速补充能量、消除疲劳，又能养颜美容、保持身材，因此水果应该是多多益善的。然而事实正好相反，科学研究表明，过量食用水果会使体内积蓄大量的维生素C，进而形成草酸。草酸和人体汗液混合排除，很容易损伤皮肤，不仅不能使皮肤细腻光滑，反而容易使皮肤变粗糙，严重时还会产生过敏性皮炎。

还有许多水果尤其不能吃过量，比如荔枝吃多了上火，香蕉吃多了容易引起体内钾、钠、钙、镁等元素的比例失调，西瓜吃多了容易引起蛋白质、矿物质摄入的减少，导致营养失衡……因此水果虽好，也不要贪吃哦~

2、彩虹原则

M身边不少的JMS都有“坚守”的好情操，比如说爱吃香蕉的天天买香蕉吃，爱吃苹果的天天吃苹果，爱吃西瓜的除了西瓜几乎都想不起别的水果来了。其实单一的水果构成并不能满足你的健康需要。

水果五颜六色的，非常好看。科学家建议，应当遵循“彩虹原则”，每天摄入3到5种水果，这样才能保证营养均衡。

红色水果：主要有番茄、石榴、草莓、樱桃等，富含类胡萝卜素，具有抗氧化作用，能够清除自由基，抑制癌细胞的形成，提高人体免疫力。而且这类水果普遍热量都很低，经常食用能让人保持轻盈体态哦！

黄色水果：这一类的代表水果主要有柠檬、木瓜、芒果、橙子、柿子、菠萝、橘子等，它们都含有丰富的天然氧化剂β—胡萝卜素，可以提高机体免疫力，还具有抗癌的功效。

紫黑色水果：主要包括李子、蓝莓、葡萄等，它们含有能消除眼睛疲劳的原花青素，这种成分还能增加血管弹性，防止胆固醇的囤积，是预防癌症和动脉硬化的最好成分。同时，紫黑色水果中钾、镁、钙等矿物质的含量也高于普通食物，它们多以有机盐的形式存在于水果中，对维持人体的离子平衡有至关重要的作用。

绿色水果：这类水果主要有青苹果、猕猴桃、鳄梨等，它们含有丰富的叶黄素或玉米黄质，有很强的抗氧化作用，还能保护视网膜免受损伤，具有保护视力的作用。

M个人的建议是，要吃多种类的水果，做成沙拉最棒咯！

「常识」三类人不宜多吃西瓜

2010年5月31日 小饭

糖尿病人。西瓜约含糖类5%，主要是葡萄糖、蔗糖和部分果糖，吃西瓜后会导致血糖增高。正常人由于会及时分泌胰岛素，可使血糖、尿糖维持在正常水平。糖尿病人则不同，在短时间内吃太多西瓜，不但血糖会升高，病情较严重的还可能因出现代谢紊乱而导致酸中毒，甚至危及生命。

肾功能不全者。肾功能不全者常会出现下肢或全身水肿那个，因为他们排出体内水分的功能很弱。若是吃太多西瓜，会因摄入过多的水又不能及时将多余水分排出而导致水分在体内超量贮存，血容量增多，不但加重水肿且容易诱发急性心力衰竭。

口腔溃疡患者。口腔溃疡在中医中被认为是阴虚内热、虚火上扰所致。西瓜有利尿作用，口腔溃疡患者多吃西瓜，会使口腔溃疡复原所需要的水分被过多排出，加重阴虚和内热，使病情绵延不易愈合。

「情调」沙拉—痛并快乐着

2010年5月24日 Michelle

无论是多么完美的女人，总会觉得自己的衣橱了少了那么一件衣服，而身上又多了那么一斤肉。减肥似乎成了女性的代名词，大部分的女生一辈子都在跟自己的体重较劲。减肥虽然意义重大，但仍旧是件影响心情的事情，改变原来的饮食习惯，把宅着玩儿电脑的时间削减出来运动……实在是让人很痛苦无比。不如苦中作乐，在单调的减肥中找点儿乐子。

天气热了，胃口小了，减肥变成了一件相对容易的事情。很多JMS都喜欢用水果来代替正餐，营养美味又能减肥，一举三得。M最近也在实行水果代餐减肥的计划，一开始还是很HAPPY的，可是渐渐就有种快要被水果淹死的痛苦感觉……这几天开始调整计划，多花一点点时间吧简单重复的水果变成好看的沙拉。

M从网上搜刮了很多很不错的沙拉图片，作为M沙拉制作的奋斗目标，在这里和大家分享一下：

M目前能做到的也就是这样的状态——Home Made，只能保证材料新鲜，颜色鲜艳，没什么技术含量，当然啦，沙拉酱总是放得特别少，而且经常用低脂酸奶代替T_T.

这样小份的沙拉非常CUTE美观，很适合下午茶的风格。

花点儿小时间把沙拉做成沙拉塔，小小地点缀一下，想象自己吃下的是特别饱满的巨无霸，减肥的过程是不是轻松点了呢？嘿嘿！

「常识」反季节水果有噱头

2010年5月6日 Michelle

M昨天从超市买了一些草莓，个子大大的看起来特别饱满可爱。洗干净了咬了一口，囧！中间是空心的不说，根本就吃不出草莓的香味，结果只好把这些草莓通通倒掉……

科技的发达让人们一年四季都能吃上心仪的水果，但是这样的反季水果真的好吗？一年四季交替，水果也原本是交替成熟的，在不同的季节吃不同的水果是我们的老祖宗几千年来的惯例。如今的化学物质或催熟或保鲜，这些催熟剂、保鲜剂、膨大剂或激素都是通过人工合成的化学物，具有一定的毒性，吃进去会加重肝肾的负担，经常食用还会损伤肝肾功能。许多人工激素还会干扰人体正常的内分泌功能，影响生长发育或产生其他疾病，因此反季水果有噱头，挑选食用需谨慎！

M整理了一些应季的水果，在这里和大家分享一下，如果筒子们有什么修改或补充记得留言告诉我哦！

春季水果：枇杷 草莓 香瓜 桃子 樱桃 李子 杨桃

夏季水果：菠萝 西瓜 荔枝

秋季水果：梨  苹果 卢柑 枣子 葡萄 香蕉 猕猴桃 木瓜

冬季水果：桔子 甘蔗 橙子

小贴士：

挑选好水果要清洗后才能放心食用，M在这里告诉大家几个小妙招，轻松洗水果，放心吃水果：

1、浸湿水果，在手上倒一点点盐，握着水果轻轻地搓，然后再用清水洗净即可。

2、将牙膏涂在水果的表面当清洁剂，这样可以把水果表皮的保鲜剂洗掉。

3、用热水洗水果，当然，M觉得这个方法比较适合冬天，嘿嘿！

4、用小盆装水，加入少许淀粉，然后放入水果进行清洗，这种方法特别适合洗葡萄，而且据说这样洗出来的水果不易变色哦。

5、到超市选购专业洗水果的洗液，注意挑选正规厂家生产的质量好的产品，千万不能使用劣质产品，因为两种化学产品混在一起危害更大哦！
