「常识」小习惯 大暖身

2009年12月27日 Michelle

M所在的城市迎来了今年的第一场雪，初雪的心情总是格外地明朗。从被窝里露出两只小眼睛看着窗外银装素裹的世界，感觉特别温馨……和很多JMS一样，M也有手冷脚冷的小毛病，如何在寒冷的冬天让身体暖和起来，只要几个饮食上的小习惯就可以实现哦！

1.多食用热性食物，在膳食中补充产热营养素。碳水化合物、脂肪、蛋白质等对提高肌体的低温耐受力都有很好的效果。代表食物：牛肉、羊肉、黄鳝、红枣、牛奶、豆类等。

食谱推介：枣杞补气茶、胡萝卜炖羊肉、干煸鳝鱼、鲫鱼豆腐汤

2. 多食用富含蛋氨酸的食物。科学研究表明，蛋氨酸可以通过转化作用形成人体耐寒所需要的甲基。寒冷的气候使得人体尿液中肌酸的排除量增加，脂肪的代谢加快，甲基则是人体肌酸合成和脂肪代谢的必要条件。代表食物：芝麻、葵花籽、叶类蔬菜。

食谱推介：杏仁黑芝麻油麦菜、芝麻羊肉、芝麻菠菜、蜜汁芝麻牛蒡丝

3. 多食用含铁质的食物。铁质是造血的重要原料，缺铁性贫血的JMS就相对更容易手脚冰凉、肢体畏寒。因此应当多补充一些含铁质的食物。代表食物：动物内脏、木耳、山药、菠菜、鸡蛋等。

食谱推介：菠菜猪肝汤、枸杞山药煲龙骨、木耳冬笋炒虾球、菠菜炒鸡蛋

4. 多食用补充碘的食物。科学实践证明，碘元素可以促进甲状腺素的分泌，它能加速体内细胞的氧化，增加身体的产热能力，加强基础代谢，加快血液循环，从而达到保暖御寒的功效。代表食物：海带、紫菜、海蜇、玉米、发菜等。

食谱推介：海带排骨汤、金钱发菜、海蜇炖猪肺、甜玉米山菌排骨汤

5. 多食用含钙的食物。钙的补充看似和御寒没有关系，其实不然。钙在人体中的含量可直接影响到人体的心肌、血管及肌肉的伸缩性和兴奋性，因此在持续的低温下要注重钙质的补充哦。代表食物：牛奶、豆类、虾皮、木耳、发菜、芝麻等。

食谱推介：姜撞奶、花生黄豆红枣汤、黄豆炖猪蹄、虾皮香芹燕麦粥

小贴士：

很多人都有喝热饮保暖的习惯，但是喝温度过高的饮料容易造成皮肤粘膜的伤害，甚至会引发食管癌。除了饮食上的小习惯之外，还要特别注意保暖。很多JMS始终坚持“先风度后温度”的基本原则不动摇，这是错误的想法，千万不要为了一时的美丽而影响了身体健康。尤其要注意下半身的保暖，睡觉前应用热水泡脚，还可以加入姜片一起泡。另外，适当的运动不光可以让身体暖和起来，也可以减少脂肪的囤积，保持曼妙曲线哦！

「常识」谁动了我的营养？

2009年12月10日 Michelle

上大学的时候，每天的早餐是千篇一律的鸡蛋、油条和豆浆。直到某一天，某JM惊呼：“呀~ 鸡蛋和豆浆在一起吃会中毒的！”M当时心里一慌，剥了壳的鸡蛋掉到了地上。鸡蛋和豆浆一起吃真的会中毒吗? 营养学家研究发现：鸡蛋和豆浆的组合会降低人体对蛋白质的吸收。因为豆浆中含有胰蛋白酶抑制物，它会抑制人体蛋白酶的活性，影响蛋白质在人体内的消化和吸收。

原来鸡蛋和豆浆一起吃不会中毒，但是是对营养的极大浪费！那么在日常饮食当中，又是哪些食物搭配，悄悄地偷走了你的营养呢？M整理了一些常见食材的禁忌搭配，在这里一起分享一下：

1. 牛奶＋菠菜/橘子/巧克力/药

像韭菜、菠菜和橘子这一类草酸含量比较多的食物，不能与牛奶同食，因为它们所含有的草酸一旦与牛奶中的蛋白质相遇会发生凝固，影响牛奶的消化和吸收。因此最好间隔2个小时以上哦。

其实很多JMS不知道，巧克力中也含有丰富的草酸，如果与牛奶中的钙质结合会形成不溶性草酸钙，它会在一定程度上阻碍钙质的吸收，对骨骼的生长也有影响，甚至导致头发干枯或者腹泻。

有人喜欢用牛奶服药，这是错误的，牛奶会明显地影响人体对药物的吸收，因此如果想喝牛奶的话，应与服药时间间隔1小时以上。

M强烈推荐牛奶可以和蜂蜜一起食用哦。德国营养学会临床实验证明：蜂蜜中含有葡萄糖、果糖、蛋白质、酶、维生素和多种矿物质，常吃可以提高人的免疫力。蜂蜜与牛奶搭配食用，能起到最佳的互补效果。最有力的证据就是德国前总理施罗德向媒体透露，他的标准早餐就是：牛奶+蜂蜜＋面包。

2. 豆腐＋菠菜/葱/竹笋

这大概是知晓度最高的禁忌搭配了。豆腐不能与含草酸高的食物一起吃，因为豆腐含钙高，钙与草酸可生成难溶或不容的草酸钙，不光影响钙的吸收，还容易产生结石。

豆腐和海带的组合则是“绝配”。因为豆腐里的皂角苷成分虽然能够促进脂肪的新陈代谢，但是容易造成机体碘缺乏，与海带同食则可以有效避免这个问题，达到优势互补。M在此推荐豆腐海带汤。

 3. 胡萝卜＋白萝卜/酒

大家还记得赵丽蓉老师的小品《打工奇遇》里的太后大酒楼么？里面有一道菜叫“群英荟萃”，被赵丽蓉老师戏称为“萝卜开会”。这道菜看起来颜色很不错，可是将胡萝卜和白萝卜一起烹调是错误的选择，因为白萝卜中维生素C的含量很高，而胡萝卜中则含有一种对抗维生素C的分解酶，可破坏白萝卜中的维生素C，所以把这两种萝卜放在一起就从直接从“开会”转为“内战”了哦。最后的结果就是其中一种萝卜的营养价值大打折扣。

胡萝卜也不可以和酒一起食用，因为胡萝卜中富含的胡萝卜素如果与酒精一同进入人体，容易在肝脏中形成毒素，导致肝病。

胡萝卜和肉一起会大大提高它的营养价值哦。在寒冷的冬天吃上一道鲜嫩美味又柔和温补的胡萝卜炖羊肉是不错的选择哦。

 4. 南瓜＋醋/羊肉/虾/螃蟹/鲤鱼/富含维生素C的食物

如果南瓜和醋同时食用，醋酸会破坏南瓜的营养成分。

至于南瓜和羊肉，早在《本草纲目》中就有记载：“南瓜不可与羊肉同食，令人气壅。”因为南瓜补中益气，羊肉大热补虚，两补同时，令人肠胃气壅，长期食用还容易诱发维生素B1缺乏症，导致黄疸和脚气病。

南瓜和虾的同食会引起痢疾，一不小心中招了，最简便的方法是用黑豆、甘草解毒。

南瓜和螃蟹或者鲤鱼的组合也是相克的，会引起食物中毒。

另外南瓜与富含维生素C的食物也是相克的，因为维生素C会被南瓜中的分解酶破坏。

M特别推荐南瓜牛肉汤，它具有润肺消痈、祛毒排脓的功效，不可小视哦。

5. 海鲜＋啤酒/水果/茶

大部分人都知道吃海鲜的时候不宜饮酒，因为海鲜在人体中代谢后会形成尿酸，尿酸过多会引起痛风、肾结石等症，而啤酒会加速体内尿酸的形成。

许多水果中含有一种叫鞣酸的物质，如苹果、柿子、葡萄、石榴、山楂等等，我们都知道海鲜里含有丰富的蛋白质和钙，鞣酸不但会降低蛋白质的营养价值，还会与海鲜中的钙质结合形成一种不易消化的物质，引起肠胃不适。

当然吃完海鲜也不能喝茶哦，原理与水果的相似，因为茶叶里也含有鞣酸。如果想喝的话，请间隔2个小时以上哦。

 6. 茶叶＋蛋

茶叶蛋是中国传统美食，M的家乡还有立夏吃茶叶蛋的习俗呢。可是茶叶蛋不宜长期食用，因为茶叶中含有生物酸碱成分，在烧煮时会渗透到鸡蛋里，与鸡蛋中的铁元素结合；这种结合体，对胃有很强的刺激性，久而久之，会影响营养物质的消化吸收，不利于人体健康。

 走出食材搭配的误区，希望大家都能吃出美味，吃出健康！

「常识」告别坏心情 你准备好了吗？

2009年11月10日 Michelle

还记得《浪漫满屋》里带来甜蜜心情的冰激凌吗？对于心情不好时吃的食物，JMS总是有说不完的选择：巧克力、冰激凌、薯片、辣椒……

选择对的食物真的能让我们心情变得更好吗？食物真的拥有让人快乐的化学元素吗？

《好心情饮食》的作者–著名营养学家苏珊克莱纳博士的回答是：“当然，我们的感受、我们应对压力时候的表现、我们怎样去思考，这些都会受到我们吃的食物的影响。”在访谈中她推荐了五类让人心情愉悦的食物：

1. 富含蛋白质和omega-3不饱和脂肪酸的食物：所有的鱼类、蟹、虾等。

科学实验证明：吸收omega-3不饱和脂肪酸越多，大脑中和心情有关的三个区域杏仁、海马和扣带中的灰质就越多，而这有助于缓解抑郁情绪。

2. 碳水化合物：谷物、豆类、水果和蔬菜等。

碳水化合物之所以能起镇静和安慰神经的作用，是由于它能够提高血清素的水平。减少疲劳感和抑郁症状。

3. 富含色氨酸的食物：牛奶及乳制品等。

色氨酸被人体吸收后，能合成神经介质5-羟色胺，它就像身体里的“信使”，有效发挥调节作用，使心情变得平静、愉快。

4. 抗氧化剂：水果、蔬菜、葡萄汁、绿茶、乌龙茶等。

氧化作用常常侵害体内细胞，造成衰老、皮肤黑斑、过敏及心血管疾病，而多吃提供抗氧化剂的食物就可以减弱体内氧化作用的程度。

5．帮助三叉神经分泌血清素的食物：爆米花、苹果、胡萝卜、芹菜等。

这一类食物的共同特点就是吃起来的时候嘎吱嘎吱的，这样可以帮助我们的三叉神经分泌出有益于好心情的血清素，而且在心情烦躁的时候吃些坚硬、清脆的食物确实是不错的选择。

小贴士：

TO GIRLS：好多JMS都喜欢吃巧克力来舒缓心情，可是摄取过度的咖啡因反而会导致人的心情沮丧，加重抑郁症状，而且吃过多的巧克力是对“减肥大业”的极大犯罪哦~

TO BOYS：很多男生在遭遇坏心情的时候往往都不愿意与人分享，大部分人会选择香烟和酒精来麻醉自己。的确，香烟和酒精可以让抑郁情绪在短时间内得到解脱，但是当它们短暂的魔法过去之后，糟糕的情绪会来得更加猛烈的！

M觉得，告别坏心情，亲自下厨做一顿饭也是不错的选择呢，看着那些新鲜的食材乖乖地服从分配，然后在自己的手中变成不可思议的美味，带来的成就感是无以言喻的……

「技巧」牛奶袋的妙用

2009年10月5日 小猪


大部分人喝完牛奶，袋子很自然地就扔掉了，今天小猪介绍一下它简单的妙用。
平时大家肉买多了，通常都要放进冷冻室里面，但是普通的塑料袋和肉经常冻在一起以至于很难掰开。这时候如何用原来装牛奶的口袋，这个问题就可以避免了：

1 买回肉之后根据不同用量把肉切成不同的小块。
2 把每次食用的量分别装在不同的牛奶口袋里面，放入冷冻室。

平时喝完牛奶，可以把牛奶袋收集到厨房，还有别的类似的包装袋，比如方便面的袋子也有一样的效果。这样既有效利用了原本要废弃的包装袋，肉也不会像用普通的塑料袋一样紧紧地冻在一起。

「常识」煮粥加牛奶营养更容易吸收

2009年9月29日 小猪

牛奶营养丰富，煮粥时也可以用牛奶使得粥的营养更加丰富，也有利于人体对营养成分的吸收。

具体的做法：
1 普通米粥：等黑米、大米或粳(jing、一声)米等主料煮得快熟时，放入适量的牛奶和糖搅拌均匀即可；
2 玉米面粥：将牛奶煮热放入适量的玉米面，搅拌均匀后放入少许奶油和糖调味，煮沸即可，这样煮出来的牛奶玉米粥味道香甜又营养丰富，特别适合婴儿食用。
3 苹果牛奶燕麦粥：将牛奶煮热，加入适量燕麦，把苹果或者香蕉切成小块放入牛奶里面，继续煮一小会。适合上班族，当早餐或者临睡前喝，营养丰富操作简单。

PS：牛奶不适宜煮沸太久，沸腾时间太长的话，会破坏其中的营养成分。

「常识」秋季干燥，蜂蜜最好

2009年9月24日 小饭

蜂蜜最佳饮用时间是一年之中在于秋，一天之内在于晨。

秋季干燥，蜂蜜最好。《本草纲目》里提到蜂蜜能清热、补中、解毒、润燥、止痛，所以非常适合秋季饮用。

早餐牛奶+蜂蜜最营养。蜂蜜的最佳饮用时间是早上，但不能空腹喝，否则会造成腹泻，特别是脾胃虚寒的人。蜂蜜与牛奶的搭配，能起到最佳的互补效果。蜂蜜作为单糖，含有较高的热能，可直接被人体吸收；而牛奶尽管营养价值较高，但热能低，单饮不足以维持人体正常的生命活动。用牛奶加蜂蜜做早餐，人体不仅能够吸收足够的热能，所补充的维生素、氨基酸、矿物质等健康物质也更全面，可以让人整个上午都精神充足。

如何挑选蜂蜜。百度给我们最详细的解答，请看这里。
