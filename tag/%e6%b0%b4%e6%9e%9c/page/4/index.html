「常识」水果怎么吃？（究极版）

2010年6月2日 Michelle

这两天M看到一则新闻，说最近几天辽宁省鞍山市几家医院连续接诊了因为空腹吃山楂、柿子而在胃里形成结石的病患。而且其中的一位女士的经历真的只能用“OMG”来形容。她曾空腹吃了两个大柿子，仅仅只过了三四天，这位女士的胃里就已经形成了一个蛋黄大小的胃石。

在如今“睡觉死”、“喝水死”、“冲凉死”屡现江湖的年代，吃个水果居然也能折腾到医院去，M不经要小小地感叹一下。其实，大家都知道水果的好处，丰富的矿物质、维生素、纤维素、不易发胖的天然糖分以及爽滑的口感都让人欲罢不能。可是水果要什么时候吃，如何选择适合你的水果，很多筒子们确实不知道的。

夏天来临，在这个水果当道的季节，M特备整理了一些水果的小常识，和大家一起分享一下水果要怎么吃。

选择恰当时机

从去年起物价就开始明显上涨，蔬菜都进入“元时代”了，水果更是涨价涨得惊人，M所在的城市普通的黄瓜涨到1.5元一斤，这还是平均价，大部分时候几乎有增无减，水果黄瓜更是涨到2.8元。所以，选择恰当的时候吃水果，让水果的营养和充分的吸收，这才是物价上涨时期的明智之举。

很多人把水果当做饭后甜点吃，其实这样做实在没什么道理，反而不利于消化和营养的吸收。吃水果的黄金时期是两餐之间，这样可以使营养收益最大化。因为水果的养分是水溶性的，容易被吸收，如果在两餐之间吃水果，肠胃可以很快地消化吸收。想减肥的JMS注意了，饭前吃水果可以让胃部产生满足感，这样吃正餐的时候就会自然地减少摄入量。不过注意咯，有些水果比如开头提到的柿子和山楂是不能空腹吃的哦！

不能空腹吃的水果

1、山楂、橘子

很多人认为山楂、橘子这一类水果酸酸甜甜的很开胃，适合饭前吃，这是错误的认识。因为其中含有大量的有机酸、果酸，具有行气消食的作用，如果空腹食用，会使胃酸猛增，对胃粘膜造成不良刺激，引发胃胀满、泛酸，还会增加饥饿感加重胃病。

2、柿子、番茄

柿子和番茄中含有丰富的柿胶粉、胶质、果胶、鞣酸和鞣红素等物质，具有很强的收敛作用。平时的时候吃对身体是没有影响的，但在空腹的状态下，胃里胃酸分泌较多，容易和这些收敛物质融合形成难以溶解的硬块，小硬块可以随着新陈代谢排除体外，但是如果形成了较大的硬块，就容易引起胃柿结石症。

3、香蕉

M曾经看过一本减肥杂志上说早餐只吃香蕉和水瘦身效果明显，并且亲自尝试过一段时间，结果吧胃折腾得奇奇怪怪的，现在想起还是内牛满面啊！香蕉并不适合空腹食用，因为香蕉中含有丰富的糖和纤维素，但在空腹时，肠胃里几乎没有可供消化的食物，这个时候吃香蕉会加快肠胃运动，促进血液循环，增加心脏负荷，严重的时候还会导致心肌梗塞呢。

4、甘蔗、荔枝

这类水果糖分很高，空腹食用会刺激胃粘膜，导致胃痛、胃胀。据说曾经有人在空腹时吃荔枝突然昏迷，医生说是体内突然渗入过量高糖分而引发的“高渗性昏迷”。专家建议，在吃荔枝前后，可以先喝一点盐水、凉茶、绿豆等，这样既不会引发昏迷，也不容易上火哦！

5、菠萝

菠萝里含有丰富的蛋白分解酵素，如果空腹吃，很容易造成胃壁受伤。像M一样喜欢吃菠萝的筒子们不如时刻念叨一下，千万不要空腹吃哦！

选择适合你的水果

很多食物都有酸碱性之分，水果也不例外，大部分的水果都属于亚酸性的，它们的酸碱性取决于水果中所含矿物质的类型。富含粗纤维、钾、钠、钙、镁的水果是低酸性水果，富含氯、磷、硫的水果则是高酸性水果。

高酸性水果：李子（PH值2.5）、杨梅（PH值1.5）、柠檬（PH值2.0）、橙子（PH值2.8）、草莓（PH值3.0）、莲雾（PH值2.8）

中酸性水果：梨子（PH值3.5）、木瓜（PH值4.5）、苹果（PH值4.9）、葡萄（PH值4.5）、桃子（PH值4.2）、菠萝（PH值4.0）、番茄（PH值4.8）

低酸性水果：香蕉（PH值5.1）、西瓜（PH值6.6）、荔枝（PH值5.0）、石榴（PH值5.0）、水蜜桃（PH值6.8）

除了酸碱值，我们在挑选水果的时候还要考虑自己的体质哦。体质燥热的筒子们适合吃梨子、香蕉、西瓜、香瓜等偏寒性水果;体质偏寒者宜吃荔枝、龙眼、樱桃、栗子等;心肌梗死、中风的病人宜吃香蕉、橘子、桃等帮助消化的水果，不宜吃柿子、苹果等含鞣酸较多的水果;冠心病、高血脂病人宜吃山楂、柑橘、柚子等含维生素C和尼克酸的水果，能降低血脂和胆固醇。

应季的才是最好的

科技的发达让人们一年四季都能吃上心仪的水果，但是这样的反季水果真的好吗？一年四季交替，水果也原本是交替成熟的，在不同的季节吃不同的水果是我们的老祖宗几千年来的惯例。如今的化学物质或催熟或保鲜，这些催熟剂、保鲜剂、膨大剂或激素都是通过人工合成的化学物，具有一定的毒性，吃进去会加重肝肾的负担，经常食用还会损伤肝肾功能。许多人工激素还会干扰人体正常的内分泌功能，影响生长发育或产生其他疾病，因此反季水果有噱头，挑选食用需谨慎！

春季水果：枇杷 草莓 香瓜 桃子 樱桃 李子 杨桃

夏季水果：菠萝 西瓜 荔枝

秋季水果：梨 苹果 卢柑 枣子 葡萄 香蕉 猕猴桃 木瓜

冬季水果：桔子 甘蔗 橙子

食用水果应遵循的原则

1、不过量原则

很多筒子们都认为水果既能迅速补充能量、消除疲劳，又能养颜美容、保持身材，因此水果应该是多多益善的。然而事实正好相反，科学研究表明，过量食用水果会使体内积蓄大量的维生素C，进而形成草酸。草酸和人体汗液混合排除，很容易损伤皮肤，不仅不能使皮肤细腻光滑，反而容易使皮肤变粗糙，严重时还会产生过敏性皮炎。

还有许多水果尤其不能吃过量，比如荔枝吃多了上火，香蕉吃多了容易引起体内钾、钠、钙、镁等元素的比例失调，西瓜吃多了容易引起蛋白质、矿物质摄入的减少，导致营养失衡……因此水果虽好，也不要贪吃哦~

2、彩虹原则

M身边不少的JMS都有“坚守”的好情操，比如说爱吃香蕉的天天买香蕉吃，爱吃苹果的天天吃苹果，爱吃西瓜的除了西瓜几乎都想不起别的水果来了。其实单一的水果构成并不能满足你的健康需要。

水果五颜六色的，非常好看。科学家建议，应当遵循“彩虹原则”，每天摄入3到5种水果，这样才能保证营养均衡。

红色水果：主要有番茄、石榴、草莓、樱桃等，富含类胡萝卜素，具有抗氧化作用，能够清除自由基，抑制癌细胞的形成，提高人体免疫力。而且这类水果普遍热量都很低，经常食用能让人保持轻盈体态哦！

黄色水果：这一类的代表水果主要有柠檬、木瓜、芒果、橙子、柿子、菠萝、橘子等，它们都含有丰富的天然氧化剂β—胡萝卜素，可以提高机体免疫力，还具有抗癌的功效。

紫黑色水果：主要包括李子、蓝莓、葡萄等，它们含有能消除眼睛疲劳的原花青素，这种成分还能增加血管弹性，防止胆固醇的囤积，是预防癌症和动脉硬化的最好成分。同时，紫黑色水果中钾、镁、钙等矿物质的含量也高于普通食物，它们多以有机盐的形式存在于水果中，对维持人体的离子平衡有至关重要的作用。

绿色水果：这类水果主要有青苹果、猕猴桃、鳄梨等，它们含有丰富的叶黄素或玉米黄质，有很强的抗氧化作用，还能保护视网膜免受损伤，具有保护视力的作用。

M个人的建议是，要吃多种类的水果，做成沙拉最棒咯！

「常识」夜生活也要补充能量

2010年5月16日 Michelle

繁华都市是一个越夜越美丽的地方。下班后，每天的美好生活才刚刚开始：下了班，约朋友见见面、唱唱歌，偶尔出去KK歌、泡泡吧……醉人的轻摇滚慢慢地从道路旁的PUB里传出来，告诉你里面有秀色和美酒，可以把你从一天的枯燥劳累里解放出来。

筒子们的夜生活大多都是唱歌、跳舞，都是能量消耗型的。如果活动进行到很晚，没有补充能量，第二天又赖床不吃早饭，那么第二天的工作效率就会大打折扣，长此以往对身体技能会有很大影响。如果空腹喝酒，没吃任何食物，那对健康的影响更大。因为酒的能量高，会影响人体对各种营养素的吸收，很容易造成机体紊乱。而且喝多了啤酒也会引起身体上的某些不适，因为每升啤酒可提供400千卡的热量，相当于200克的面包，或500克的土豆，或60克的奶油……把这些“定时炸弹”吃下去，不光是健康的大敌，也为肥胖埋下了隐患哦。

夜生活也要讲究营养的补充，但是不宜随意或者过度补充，恰当、适度是前提。M整理了一些夜生活能量补充小贴士，在这里和大家一起分享一下：

1、可以吃一点甜点，但不宜太多，50~100克为佳。

2、清淡的粥和煲汤最好，又填肚子又养胃。

3、不要吃高热量或者高蛋白的食物，如薯条、爆米花、方便面、烤肉等，这些食物不容易消化又容易囤积脂肪。

4、可以适当吃一些水果或是沙拉，但不要吃糖分较高的水果如葡萄等，否则太兴奋了不宜睡着。

5、不要吃得太咸，这样会疯狂喝水，最后造成第二天面部水肿。

6、如果去夜市摊子要注意卫生哦，如果吃了脏东西闹肚子就得不偿失了。

7、夏天到了，不要在夜晚吃太多冰品，容易刺激肠胃，引起肠胃不适。

「常识」不宜生吃和最好生吃

2010年5月7日 小饭

健康和营养均衡是现代人饮食所关注的两大重点。不少商家利用生食作为营销噱头，生食这把双刃剑带给人们的利与弊引发了餐桌上的激烈讨论。生食是否真的有营养？生食又会带给人们哪些疾病呢？哪些食物不宜生吃哪些食物最好生吃，小饭做了以下整理和归类，欢迎补充、指正。

以下食物不宜生吃：

肉类：河鱼、螃蟹、龙虾、羊肉、猪肉

河鱼：生吃河鱼，鱼体内的寄生虫会进入人体发育成虫，易导致胆管炎甚至肝硬化。需要说明的是这里的生吃河鱼与日本人常吃的刺身是两个概念，生鱼片主要以深海鱼为原材料，从选料、刀工、使用的砧板和装盘都有严格的质量把关，确保食物清洁无污染，并且最主要的佐料——芥末有杀菌功能。

螃蟹、龙虾：螃蟹和龙虾都带有肺吸虫，生吃会造成肺脏损伤，严重者会使肠道发炎或肠道水肿充血。

羊肉：不少人喜欢吃很嫩的涮羊肉，一般只涮到七八成熟，常吃容易感染上旋毛虫病。倘若旋毛虫进入血液，周游全身，最后定居于肌肉，可引起恶心、呕吐、腹泻、高热、头痛、肌肉疼痛以及腿肚子剧痛、运动受限等；幼虫若进入脑和脊髓，还能引起脑膜炎症状。

猪肉：猪肉绦虫与羊肉旋毛虫一样可怕……

蔬菜：荸荠、鲜黄花菜、豆角

荸荠：又叫马蹄，常吃生荸荠， 其中的姜片虫就会进入人体并附在肠粘膜上，可造成肠道溃疡、腹泻或面部浮肿。

鲜黄花菜：含有秋水仙碱，进入人体形成剧毒的氧化二秋水仙碱，食用3-20毫克就可致死。

豆角：包括扁豆、刀豆、四季豆等各种豆。豆角里面含有一种毒蛋白“凝集素”，食用后会导致中毒。豆角至少要在沸水中加热10分钟以上才能食用。

饮料：豆浆

豆浆：饮用未煮沸的豆浆会引起全身中毒。

其他：鸡蛋、木耳、白糖、蜂蜜

鸡蛋：食用生鸡蛋人体消化功能会被破坏，并且生鸡蛋常含有沙门氏菌，会使人呕吐、腹泻。

木耳：生吃新鲜木耳后可引起日光性皮炎，严重者出现皮肤瘙痒、水肿和疼痛。

白糖：白糖在生产、包装、运输和储存的过程中，很容易受到污染，尤其是存放时间过长的、颜色变黄的白糖，食用后会引起胃肠不适、腹泻等症状，甚至引起过敏反应。白糖最好加热到70摄氏度并超过3分钟再食用。

蜂蜜：蜜蜂在酿制蜂蜜时会采集到一些有毒的花粉，这些有毒的花粉酿进蜂蜜以后，人吃了生蜜就容易发生中毒。另外蜂蜜在收获、运输、保管的过程中，又很容易被细菌污染。因此，生蜂蜜不可食用。

最好生吃的食物：

水果：大部分的水果是可以生吃的并且最好生吃。调查显示，加热的过程会让水果所能提供给人体的营养成分减少一半或更多。因此生食可以使营养成分得到充分的保存，有助于提高身体的免疫力、增强抗氧化功能、排除体内有害物质，更能达到净化身体、美容养颜的作用。但是在生食水果的时候要注意选择适合自己体质的，并不是所有的水果都适合每一个身体。

蔬菜：黄瓜、番茄、

黄瓜：含有丰富的维生素和微量元素，营养丰富清脆爽口，最好是连皮一起吃。但是考虑到表皮残留农药的隐患，最好先在盐水中泡15-20分钟再洗净连皮生食。

番茄：番茄中维生素A较丰富，对视力保护及皮肤晒后修复有好处。有些人怕酸喜欢吃“雪漫火焰山”的加糖凉拌番茄，其实番茄还是原汁原味生吃最好！尤其是肥胖者、糖尿病人、高血压病人更不宜吃白糖拌西红柿。

海产品：三文鱼、黑鱼、金枪鱼等

三文鱼、黑鱼、金枪鱼等海产品中含有丰富的矿物质和微量元素，是健康的身体必不可少的。过度烹饪会破坏蛋白质中的活性酶，导致身体对食物营养的吸收大打折扣。但是小饭身边不少人生吃这类食品会导致腹泻等不良反应，所以决定是否生食还是得取决于自身体质。

生食小贴士：大蒜和醋具有杀菌功能，生食以上蔬菜和海产品提倡适量加入醋和蒜，既能调味又能杀菌。但胃酸分泌过多者尽量不加，以免刺激肠胃。

「常识」反季节水果有噱头

2010年5月6日 Michelle

M昨天从超市买了一些草莓，个子大大的看起来特别饱满可爱。洗干净了咬了一口，囧！中间是空心的不说，根本就吃不出草莓的香味，结果只好把这些草莓通通倒掉……

科技的发达让人们一年四季都能吃上心仪的水果，但是这样的反季水果真的好吗？一年四季交替，水果也原本是交替成熟的，在不同的季节吃不同的水果是我们的老祖宗几千年来的惯例。如今的化学物质或催熟或保鲜，这些催熟剂、保鲜剂、膨大剂或激素都是通过人工合成的化学物，具有一定的毒性，吃进去会加重肝肾的负担，经常食用还会损伤肝肾功能。许多人工激素还会干扰人体正常的内分泌功能，影响生长发育或产生其他疾病，因此反季水果有噱头，挑选食用需谨慎！

M整理了一些应季的水果，在这里和大家分享一下，如果筒子们有什么修改或补充记得留言告诉我哦！

春季水果：枇杷 草莓 香瓜 桃子 樱桃 李子 杨桃

夏季水果：菠萝 西瓜 荔枝

秋季水果：梨  苹果 卢柑 枣子 葡萄 香蕉 猕猴桃 木瓜

冬季水果：桔子 甘蔗 橙子

小贴士：

挑选好水果要清洗后才能放心食用，M在这里告诉大家几个小妙招，轻松洗水果，放心吃水果：

1、浸湿水果，在手上倒一点点盐，握着水果轻轻地搓，然后再用清水洗净即可。

2、将牙膏涂在水果的表面当清洁剂，这样可以把水果表皮的保鲜剂洗掉。

3、用热水洗水果，当然，M觉得这个方法比较适合冬天，嘿嘿！

4、用小盆装水，加入少许淀粉，然后放入水果进行清洗，这种方法特别适合洗葡萄，而且据说这样洗出来的水果不易变色哦。

5、到超市选购专业洗水果的洗液，注意挑选正规厂家生产的质量好的产品，千万不能使用劣质产品，因为两种化学产品混在一起危害更大哦！

「窍门」家有小淘气

2010年3月15日 Michelle

姐姐的孩子今年8岁了，上小学二年级，活泼好动可偏偏就是个小胖墩。和他一起吃饭的时候发现他只吃肉类，桌子上的清蒸鱼、红烧肉、扣肉什么的一定不会放过，可是对于青菜，就保持视而不见的态度。夹到他碗里，他不是顽皮地丢掉就是夹给爸爸吃，好不容易“逼”他吃下去了，咀嚼的时候却露出一副服毒自尽的样子……终于奶奶看不下去了，抱着“孩子还小，长大了就懂事了”的溺爱心理又帮他“挡过一劫”……

曾经还有过这样的对话：

——你喜欢喜羊羊还是灰太狼呢？

——我喜欢喜羊羊！

——那你知道喜羊羊喜欢吃什么吗？

——（发现中计，立马改口）那我喜欢灰太狼！！

囧……

你家的小朋友是否也有这样的小淘气呢？很多家长都在抱怨自己的孩子不爱吃蔬菜，于是到药店里买了价格不菲的各类维生素片。但事实上，水果蔬菜中丰富的维生素群绝非是简单的小药片可以取代的。盲目补充维生素也不能保证孩子是否能有效吸收。

科学研究证明：蔬菜中丰富的钾能够起到镇静安神的作用，并且能够使体内摄取过量的钠排除体外。不爱吃蔬菜的孩子，体内多余的钠往往不能尽数排出，残留在体内就容易诱发焦虑和不稳定情绪。加上动物性食物及食盐、味精、小苏打中的钠也会促进神经兴奋，那么孩子就容易产生多动好动、注意力不集中的现象。由此可见，小淘气们不爱吃蔬菜会是多么严重的问题啊！

M小时候也不爱吃蔬菜，尤其不爱吃胡萝卜。于是M的妈妈就把胡萝卜剁碎在肉泥里，包成香喷喷的肉包子，或者在榨果汁的时候放进半根，量掌握得好味道不错的话小孩子是难以察觉的。体察妈妈们的苦心，M特别整理了一些将蔬菜隐秘加工的美味菜谱，在这里和大家分享一下：

缤纷土豆泥（超市里有很多土豆泥，但都没有妈妈厨房里做的好吃，因为妈妈厨房里的东西最安全、有爱心、不添加任何防腐剂！^o^）

原料：土豆、香肠、黄瓜、胡萝卜、紫甘蓝、沙拉酱

1、土豆洗净去皮，切片蒸熟；

2、香肠、黄瓜、胡萝卜切丁，紫甘蓝切丝；

3、蒸熟的土豆用勺子压成泥，过筛口感会更细腻哦；

4、土豆泥中拌适量沙拉酱，调少许盐和鸡精；

5、将准备好的菜菜放入土豆泥中混合均匀，装入裱花袋中挤成自己喜欢的样式即可。

胡萝卜金针奶酪卷（这道菜味道清淡，不需要用太多的调味料也能香气四溢，小盆友光闻着就流口水了，更别说蛋饼的淡淡香味，在口中沙沙作响的奶酪和培根，恐怕连胡萝卜的味道、韭菜的味道都抛到九霄云外去了！^o^）

原料：胡萝卜、金针菇、韭菜、香葱、鸡蛋、奶酪片、培根

1、将胡萝卜切丝，韭菜切段，金针菇切段。

2、将鸡蛋打撒，在锅上摊成薄蛋饼。

3、蛋饼上铺上奶酪片，培根片、胡萝卜丝、韭菜、金针菇卷成卷。

4、用香葱或烫过的整根韭菜将蛋皮卷困扎好，放入微波炉中用光波烤2分钟，待奶酪糊化即可取出装盘。

肉馅酿苦瓜（别说小盆友，好多筒子们对苦瓜也是深恶痛绝。花一点点时间做一道精雕细琢的苦瓜点心，用真诚而享受的神情把苦瓜通通干掉吧！^o^）

原料：苦瓜、肉馅、干香菇、姜末、淀粉、白胡椒粉、盐、鸡精、生抽、糖、湿淀粉

1，干香菇事先加水泡发，挤干水切末待用。肉馅加姜末、香菇末、淀粉、白胡椒粉和适量水拌匀，朝一个方向搅打上劲

2，将苦瓜从中间纵向剖开，用勺子挖去内瓤

3，将肉馅添在苦瓜内

4，上火蒸熟，约10分钟

5，取出切段，蒸苦瓜时溢出的汁待用

6，锅中放入生抽、糖和蒸苦瓜的汁烧开，加湿淀粉勾芡，淋在苦瓜段上即可

洋葱饺（洋葱饺除了口味独特还有不错的强身功效哦，以此为例还可以将很多其他蔬菜混进肉馅中包成饺子，美味营养又健康！^o^）

原料：猪肉、洋葱（白皮）、小麦面粉、大葱、姜、盐、味精、鸡蛋

1.将新鲜猪肉洗净绞成肉馅备用

2.将洋葱一切为二，去掉老皮和两头，先切成片，再切成末，加油、盐3克、味精和鲜肉馅后拌匀即成洋葱馅

3.将面粉、盐2克、水30毫升、鸡蛋混合在一起，揉搓成面团，再将面团分小块，再擀成面皮，备用

4.擀好的面皮中包入馅，捏好，包成饺子，以常法煮熟食之

奶油花菜蘑菇浓汤（喝惯了传统的煲汤不如换换口味，来份咸咸的“冰激凌”。光是看起来就足够勾起馋虫了，更别说弥漫在口中的浓郁香醇，绝对征服那些不爱吃西兰花的小盆友咯！^o^）

原料：西兰花、白花菜、蘑菇、黄油、牛奶、面粉、高汤、盐、黑胡椒

图片来源网络

1：将西兰花，花菜掰小朵，放入水中焯煮三分钟后捞起过凉水备用

2：平底锅开小火，黄油50克放入锅中慢慢融化，

3：融化后，将面粉慢慢筛入锅中，边筛边将面粉和黄油搅拌均匀

4：加一半牛奶小火慢慢炒成黄油炒面盛出

5：开小火，放入20克黄油融化，放入黄油炒面搅拌匀
6：在搅拌均匀的黄油炒面中加入高汤煮开

7：汤开后放入西兰花，花菜，蘑菇继续煮开，煮三分钟

8：在汤中加入牛奶，盐，黑胡椒调味即可
